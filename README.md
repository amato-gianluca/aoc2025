# aoc2025
My Rust solutions for AoC (Advent of Code) 2025.

Source files whose name starts with `chat_` are generated by ChatGPT 5.1 using the following prompt:

<blockquote>
  Read the following text and write a Rust program which solves both parts. The program should take its data from the standard input.
</blockquote>

Then the entire text of both parts follows, without the lines with the puzzle answers.
